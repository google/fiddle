Autofill: Deep Configurability
==============================

Fiddle's ``autofill`` features enable library authors to allow end users to
configure deep object trees with a minimum of boilerplate. Library authors add
a few annotations in their code using ``Autofill`` to specify the appropriate
Fiddle behavior.
